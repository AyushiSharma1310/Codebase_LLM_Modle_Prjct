

### Setup Instructions

1. **Create Virtual Environment**
    ```powershell
    python venv AI-Model\venv
    ```
    Then
    ```powershell
    AI-Model\venv\Scripts\activate

1. **Install dependencies:**
   ```powershell
   pip install -r requirements.txt
   ```


3. **Run the application:**
   ```powershell
   streamlit run app.py
   ```
